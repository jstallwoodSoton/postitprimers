Sums of Powers

General Formula

$$ a\sum_{x = 1}^{n}x^{(a - 1)} = \begin{cases} a = 1 \rightarrow (n + 1)^a - 1 \\ a \ge 2 \rightarrow (n + 1)^a - 1 - \sum_{k = 0}^{a - 2}\binom{a}{k}\sum_{x = 1}^{n} x^k \end{cases}$$
$$ \forall (a, n) \in \mathbb{N}$$

Derivation

$$ \sum_{x = 1}^{n} (x + 1)^{a} = \sum_{x = 1}^{n} \sum_{k = 0}^{a} \binom{a}{k}x^{k} $$ $$ \sum_{x = 1}^{n} (x + 1)^{a} = \sum_{x = 1}^{n} \left(\binom{a}{0}x^0 + \binom{a}{1}x^1 + ... + \binom{a}{a - 1}x^{(a - 1)} + \binom{a}{a}x^a \right) $$ $$ \sum_{x = 1}^{n} (x + 1)^{a} = \binom{a}{0}\sum_{x = 1}^{n}x^0 + \binom{a}{1}\sum_{x = 1}^{n}x^1 + ... + \binom{a}{a - 1}\sum_{x = 1}^{n}x^{(a - 1)} + \binom{a}{a}\sum_{x = 1}^{n}x^a $$ $$ \sum_{x = 1}^{n} (x + 1)^{a} - \binom{a}{a}\sum_{x = 1}^{n}x^a = \binom{a}{0}\sum_{x = 1}^{n}x^0 + \binom{a}{1}\sum_{x = 1}^{n}x^1 + ... + \binom{a}{a - 1}\sum_{x = 1}^{n}x^{(a - 1)} $$
$$ \sum_{x = 1}^{n} (x + 1)^{a} - \binom{a}{a}\sum_{x = 1}^{n}x^a = 2^a + 3^a + ... + n^a + (n + 1)^a - 1^a - 2^a - ... - (n - 1)^a - n^a$$ $$ \sum_{x = 1}^{n} (x + 1)^{a} - \binom{a}{a}\sum_{x = 1}^{n}x^a = (n + 1)^a - 1$$
$$ (n + 1)^a - 1 = \binom{a}{0}\sum_{x = 1}^{n}x^0 + \binom{a}{1}\sum_{x = 1}^{n}x^1 + ... + \binom{a}{a - 1}\sum_{x = 1}^{n}x^{(a - 1)} $$ $$ \binom{a}{a - 1}\sum_{x = 1}^{n}x^{(a - 1)} = (n + 1)^a - 1 - \binom{a}{0}\sum_{x = 1}^{n}x^0 - \binom{a}{1}\sum_{x = 1}^{n}x^1 - ... - \binom{a}{a - 2}\sum_{x = 1}^{n}x^{(a - 2)} $$
$$ a\sum_{x = 1}^{n}x^{(a - 1)} = \begin{cases} a = 1 \rightarrow (n + 1)^a - 1 \\ a \ge 2 \rightarrow (n + 1)^a - 1 - \sum_{k = 0}^{a - 2}\binom{a}{k}\sum_{x = 1}^{n} x^k \end{cases}$$

Let \(a = 1\)

$$ \sum_{x = 1}^{n}x^{0} = (n + 1)^1 - 1 $$ $$ \sum_{x = 1}^{n}x^{0} = n + 1 - 1; \text{ thus: } $$
$$ \sum_{x = 1}^{n}x^{0} = n$$

Let \(a = 2\)

$$ 2\sum_{x = 1}^{n}x^{1} = (n + 1)^2 - 1 - \sum_{k = 0}^{0}\binom{2}{k}\sum_{x = 1}^{n} x^k$$ $$ 2\sum_{x = 1}^{n}x^{1} = (n + 1)^2 - 1 - \sum_{x = 1}^{n} x^0$$ $$ 2\sum_{x = 1}^{n}x^{1} = n^2 + 2n + 1 - 1 - n$$ $$ 2\sum_{x = 1}^{n}x^{1} = n^2 + n; \text{ thus: }$$
$$ \sum_{x = 1}^{n}x^{1} = \frac{n^2 + n}{2} = \frac{n(n + 1)}{2}$$

Let \(a = 3\)

$$ 3\sum_{x = 1}^{n}x^{2} = (n + 1)^3 - 1 - \sum_{k = 0}^{1}\binom{3}{k}\sum_{x = 1}^{n} x^k $$ $$ 3\sum_{x = 1}^{n}x^{2} = (n + 1)^3 - 1 - \binom{3}{0}\sum_{x = 1}^{n} x^0 - \binom{3}{1}\sum_{x = 1}^{n} x^1 $$ $$ 3\sum_{x = 1}^{n}x^{2} = n^3 + 3n^2 + 3n + 1 - 1 - n - 3\left(\frac{n^2 + n}{2}\right) $$ $$ 3\sum_{x = 1}^{n}x^{2} = n^3 + 3n^2 + 3n + 1 - 1 - n - \frac{3n^2 - 3n}{2} $$ $$ 6\sum_{x = 1}^{n}x^{2} = 2n^3 + 3n^2 + n; \text{ thus: } $$
$$ \sum_{x = 1}^{n}x^{2} = \frac{2n^3 + 3n^2 + n}{6} = \frac{(2n + 1)n(n + 1)}{6} = \frac{(2n + 1)}{3}\sum_{x = 1}^{n}x$$

Let \(a = 4\)

$$ 4\sum_{x = 1}^{n}x^{3} = (n + 1)^4 - 1 - \sum_{k = 0}^{2}\binom{4}{k}\sum_{x = 1}^{n} x^k $$ $$ 4\sum_{x = 1}^{n}x^{3} = n^4 + 4n^3 + 6n^2 + 4n - \sum_{x = 1}^{n} x^0 - 4\sum_{x = 1}^{n} x^1 - 6\sum_{x = 1}^{n} x^2$$ $$ 4\sum_{x = 1}^{n}x^{3} = n^4 + 4n^3 + 6n^2 + 3n - 4\left(\frac{n^2 + n}{2}\right) - 6\left(\frac{2n^3 + 3n^2 + n}{6}\right)$$ $$ 4\sum_{x = 1}^{n}x^{3} = n^4 + 2n^3 + n^2; \text{ thus: } $$
$$ \sum_{x = 1}^{n}x^{3} = \frac{n^4 + 2n^3 + n^2}{4} = \frac{n^2(n + 1)^2}{4} = \left( \frac{n(n + 1)}{2}\right)^2 = \left(\sum_{x = 1}^{n}x \right)^2 $$

Let \(a = 5\)

$$ 5\sum_{x = 1}^{n}x^{4} = (n + 1)^5 - 1 - \sum_{k = 0}^{3}\binom{5}{k}\sum_{x = 1}^{n} x^k $$ $$ 5\sum_{x = 1}^{n}x^{4} = (n + 1)^5 - 1 - \binom{5}{0}\sum_{x = 1}^{n} x^0 - \binom{5}{1}\sum_{x = 1}^{n} x^1 - \binom{5}{2}\sum_{x = 1}^{n} x^2 - \binom{5}{3}\sum_{x = 1}^{n} x^3 $$ $$ 5\sum_{x = 1}^{n}x^{4} = n^5 + 5n^4 + 10n^3 + 10n^2 + 4n - 5\left( \frac{n^2 + n}{2}\right) - 10\left(\frac{2n^3 + 3n^2 + n}{6}\right) - 10\left(\frac{n^4 + 2n^3 + n^2}{4}\right) $$ $$ 5\sum_{x = 1}^{n}x^{4} = n^5 + 5n^4 + 10n^3 + 10n^2 + 4n - \frac{5n^2 - 5n}{2} - 10\left(\frac{2n^3 + 3n^2 + n}{6}\right) - \frac{5n^4 - 10n^3 - 5n^2}{2} $$ $$ 10\sum_{x = 1}^{n}x^{4} = 2n^5 + 5n^4 + 10n^3 + 10n^2 + 3n - 20\left(\frac{2n^3 + 3n^2 + n}{6}\right) $$ $$ 10\sum_{x = 1}^{n}x^{4} = 2n^5 + 5n^4 + 10n^3 + 10n^2 + 3n - \frac{20n^3 - 30n^2 - 10n}{3} $$ $$ 30\sum_{x = 1}^{n}x^{4} = 6n^5 + 15n^4 + 10n^3 - n; \text{ thus: } $$
$$ \sum_{x = 1}^{n}x^{4} = \frac{6n^5 + 15n^4 + 10n^3 - n}{30} = \frac{n(n^4 - 1) + 5n^3(n + 2)(n + 1)}{30}$$
